opencv_tests
============

Install OpenCV
--------------

[Install Instructions](../../INSTALL_XUBUNTU.md)

Build This Example
------------------

```shell
mkdir -p ~/builds/opencv_tests/opencv_tests-1.0
cd ~/builds/opencv_tests/opencv_tests-1.0
cmake ~/git/camera_computer_linux/examples/opencv_tests/
make
```

Run This Example
----------------

```shell
cd ~/builds/opencv_tests/opencv_tests-1.0
./opencv_tests
```
